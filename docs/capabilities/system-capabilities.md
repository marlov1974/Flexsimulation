# System capabilities

System capabilities are shared capabilities that create portfolio and business-case effects across assets.

## Core list

1. Customer and asset information model
2. Observable energy state
3. 15-minute plan and baseline management
4. Stacked optimisation
5. Flexibility forecasting and risk quantification
6. Portfolio virtual battery management
7. Activation orchestration
8. Rebound scheduling and intraday monetisation
9. Delivery verification and settlement
10. Business case and value attribution
11. Cross-asset portfolio effects and dependency modelling

## Strategic rule

These capabilities should not be treated as isolated vendor modules. Vendors can support implementation, but the capability definitions, value logic and optimisation goals should remain owned by the product/business architecture.
