# Outsourcing boundaries

Some capabilities are poor candidates for black-box outsourcing because they define how system value is created.

## Hard-to-outsource capabilities

- Business case and value attribution
- Customer and asset information model
- 15-minute plan and baseline management
- Stacked optimisation
- Rebound scheduling and intraday monetisation
- Flexibility forecasting and risk quantification
- Portfolio virtual battery management
- Customer promise and control rights

## Better outsourcing candidates

- device adapters
- telemetry ingestion implementation
- forecast data services
- UI components
- BSP or market-access operation
- optimisation algorithm implementation under owned objectives

## Principle

Outsource implementation acceleration, not system ownership. Do not outsource the logic that determines how integrated value is created, risk-adjusted and attributed.
