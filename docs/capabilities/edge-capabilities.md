# Edge capabilities

Edge capabilities connect specific resource types or use cases to the system capabilities.

## Initial edge capabilities

- EV smart charging to flexibility markets
- EV V2X to flexibility markets
- cabin battery flexibility
- PV forecast and curtailment
- heat pump and hot-water flexibility
- cabin heating and frost-protection flexibility
- pool or premium-cabin load flexibility
- dual-home energy sharing
- home battery optimisation
- local grid constraint handling

## Rule

An edge capability should not be defined only as technical control of an asset. It should define how the asset is observed, planned, risk-adjusted, activated, restored and value-attributed inside the system.
