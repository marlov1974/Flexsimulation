# Executive playground purpose

Flexsimulation should become an interactive executive business-case playground.

The purpose is to replace a static presentation and spreadsheet with a page where decision makers can explore assumptions directly.

## Desired user experience

Users should be able to adjust:

- number of customers,
- customer segments,
- enabled edge capabilities,
- enabled system capabilities,
- ownership model,
- market scope,
- risk level,
- customer value sharing.

The page should then show how results change:

- net value per year,
- value per customer,
- customer value,
- P95 bidable flexibility,
- rebound cost or value,
- imbalance impact,
- delivery risk,
- strategic control and stickiness.

## Intended insight

The page should not only present a conclusion. It should let the user discover that the same assets become more valuable when connected through shared system capabilities.

Core message:

```text
Edge capabilities create raw potential.
System capabilities convert potential into profitable, risk-adjusted and customer-valued portfolio outcomes.
```
