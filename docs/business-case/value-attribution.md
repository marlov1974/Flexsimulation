# Value attribution

Business-case value should be attributed to capabilities where possible.

## Initial value categories

- customer savings
- energy-sharing value
- spot optimisation value
- intraday optimisation value
- flexibility-market revenue
- imbalance cost avoided
- rebound cost or rebound value
- delivery error and penalties
- platform and operational cost
- capability uplift

## Principle

Do not only report gross flexibility revenue. Report net portfolio value after rebound, risk, customer constraints and operating cost.
