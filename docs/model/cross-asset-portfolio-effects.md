# Cross-asset portfolio effects

The integrated system can create value beyond the sum of isolated asset values.

Examples:

- better household-load forecast can free battery capacity,
- battery fallback can increase EV portfolio P95 capacity,
- PV forecast can create legitimate EV charging plans,
- EV and heat-pump rebound can be scheduled against each other,
- station batteries can hedge PV forecast error,
- residual load forecasting can reduce buffers across assets.

The simulator should compare asset-by-asset and integrated scenarios to quantify these effects.
