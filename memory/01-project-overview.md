# Project overview

Flexsimulation is a generic modelling prototype for quantifying the business value of integrated distributed energy capabilities.

The core question is:

```text
How much additional value is created when distributed customer resources are planned, risk-adjusted and monetised as one integrated virtual battery instead of being optimised asset by asset?
```

The simulator is a business-case machine first and an energy simulator second.

Primary outputs should include:

- incremental value vs fragmented optimisation,
- SEK/customer/year,
- portfolio up/down capacity by confidence level,
- rebound cost and value,
- delivery success and error,
- value attribution by capability.

The simulator should use synthetic customers, synthetic assets, synthetic prices and synthetic activation events unless explicit public-safe data is introduced later.
